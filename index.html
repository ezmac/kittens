Kittens code

There's this game called Kittens game. Yeah, this is code to automate that. Why? It seemed like fun to automate it. It's single player only, so it doesn't matter. It was good practice in javascript.

to use, hit ctrl shift I (command shift I on mac) or get into web dev tools somehow and paste this into the console there. Read the script first if you're paranoid. oh, and hit the trade button first. it's a bug I still haven't fixed.
